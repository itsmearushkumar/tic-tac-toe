# 🎮 Tic Tac Toe Web App

A modern Tic Tac Toe game built with HTML, CSS, and JavaScript.

## 🌟 Features

- 👥 Player vs Player mode
- 🤖 Play vs AI (Minimax Algorithm)
- 🎯 Smart AI (unbeatable)
- 💡 Clean and modern UI
- ✨ Hover effects and animations
- 📊 Live scoreboard
- 🔄 Auto restart after game ends

## 🌐 Live Demo

👉 https://itsmearushkumar.github.io/tic-tac-toe/

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🧠 AI Logic

The AI uses the **Minimax Algorithm** to make optimal decisions and never lose.
